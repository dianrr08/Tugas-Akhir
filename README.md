## 1. User Guide Dokumentasi
Dokumentasi yang disusun adalah **User Guide** yang bertujuan membantu pengguna dalam memahami dan menggunakan aplikasi To-Do List.

## 2. Tujuan dan Target Audiens

### Tujuan
- Memberikan panduan penggunaan aplikasi
- Membantu pengguna memahami fitur utama
- Mengurangi kesalahan penggunaan

### Target Audiens
- Pengguna umum (non-teknis)
- Pengguna pemula aplikasi
